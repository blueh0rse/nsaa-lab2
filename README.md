## Setup

1. Clone the repository

```batch
git clone https://github.com/blueh0rse/nsaa-lab2
```

2. Initiate npm project

```batch
npm init
```

3. Install dependencies

```batch
npm install
```

4. Start the server

```batch
npx nodemon index.js
```

5. Access the web page

```batch
http://localhost:3000/
```
